# github-user-app-dicoding-submission
Last submission of "Belajar Fundamental Aplikasi Android" class from Dicoding

Features  
1. Search username
2. Display user detail
3. Display list of follower dan following
4. Add user to favorite
5. Localization (English and Indonesian)
6. Dark Mode
7. Share user detail
